# Cognito - Intelligent Attendance System

**Cognito** is a real-time facial recognition attendance system.

## Setup
1.  Install dependencies: `pip install -r requirements.txt` (or use the mock mode provided).
2.  Run the app: `python app.py`
3.  Open `http://127.0.0.1:5000`

## Features
-   Real-time Face Recognition
-   Admin Dashboard
-
   Firebase Integration
